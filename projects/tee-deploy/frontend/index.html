Getting Started

1. Create a Telegram bot via @BotFather

2. Get an API key from your AI provider (Anthropic, OpenAI, etc.)

3. Find your Telegram ID via @userinfobot

4. Click SPAWN ‚Äî that's it

What's a TEE?

A Trusted Execution Environment is a hardware-secured enclave. Your bot's API keys and data are encrypted by the CPU itself ‚Äî not even the server operator can read them. Learn more ‚Üí

What's OpenClaw?

OpenClaw is the open-source AI agent framework with 189K GitHub stars. Agents have persistent memory, skills, and a soul. Clawster deploys them into secure enclaves so they can run 24/7 without trusting anyone ‚Äî not even us. Read the docs ‚Üí
